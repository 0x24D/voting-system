# voting-system

## Technologies used
* Frontend (Vue)
* Middleware (Express)
* Backend (MongoDB)
* Written using ES6 and running on node.js server
* Tested using Mocha and Chai

## API
* Create - POST
* Read - GET
* Update - PUT
* Delete - DELETE
```
/api/v1/addresses
/api/v1/campaigns
/api/v1/candidates
/api/v1/constituencies
/api/v1/pollingStations
/api/v1/systems
/api/v1/users
```
